   - Web Campでは2スペース
- 要素
   - 開始タグ
   - コンテンツ
   - 終了タグ
- HTMLの構成
    - ルート要素
    - ヘッダー要素
    - メイン要素
    - フッター要素
- 文書型宣言
    - `<!DOCTYPE HTML>`
- 空要素
    - `<br>`
    - `<img>`
    - `<input>`
    - `<meta>`
    - `<link>`
    - `<hr>`  ...etc
- 属性
    - 属性の構成
        - 属性名
        - 属性値

    - `id`
    - `class`
    - `style`
    - `title`
    - `lang`
    - `data-`
    - `hidden` ...etc
